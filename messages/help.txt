Сообщение с капчей - {correct_answer}
Сообщение со ссылкой и статистикой - {stars_count} {ref_count} {ref_url}
Сообщение с "Заданием" - {task_str}
Сообщение после капчи - не содержит переменных

Описание переменных:
{correct_answer} - переменная, которая содержит в себе правильный вариант для каптчи

{stars_count} - переменная выводит общее кол-во звезд для каждого пользователя

{ref_count} - переменная содержит общее кол-во приглашенных пользователей прошедших каптчу

{ref_url} - сама реферальная ссылка пользователя

{task_str} - переменная содержит информацию о конкретном задании и имеет следующий вид:

```
Каналы для подписки:

https://t.me/was_is_dass

Вознаграждение: 2.0
```
